O que é o TDIA-CodeGen

Um ambiente dedicado à Engenharia de Dados para escrever, testar e versionar queries SQL sobre os DataMarts e DiamondLayers da plataforma — e gerar, a partir delas, código Python pronto para os seus pipelines.

Como vai funcionar

  1. Selecione o DataMart e a DiamondLayer de trabalho.
  2. Escreva a query no editor SQL (com autocomplete do catálogo).
  3. Execute e visualize os resultados na grade.
  4. Exporte em CSV, Excel ou JSON.
  5. Gere código Python (pandas / pyspark / SQLAlchemy) para rodar em dataflow ou VM.

Quem acessa

  • Root
  • Administrador
  • Engenheiro de Dados

Boas práticas

  • Use LIMIT ao explorar dados.
  • Qualifique nomes com aspas quando houver maiúsculas/espaços.
  • Prefira ILIKE para texto sem distinção de caixa.

Exemplos

Clique em "Usar no editor" e ajuste para as suas tabelas (o autocomplete sugere as tabelas/colunas do seu escopo).

SELECT *
FROM sua_tabela
LIMIT 20;
SELECT categoria, COUNT(*) AS qtd, SUM(valor) AS total
FROM sua_tabela
GROUP BY categoria
ORDER BY total DESC
LIMIT 20;
CREATE TABLE minha_tabela_tech AS
SELECT a.id, a.nome, b.valor
FROM tabela_a a
JOIN tabela_b b ON b.a_id = a.id;
